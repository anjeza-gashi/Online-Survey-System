# Online Survey System

A backend API to create, manage, and collect responses for surveys. Built with **Node.js** and **MongoDB**.

## Features

- **Survey Management**: Create, read, update, and delete surveys  
- **Question Management**: Add, update, and delete questions in surveys  
- **Responses**: Submit survey responses anonymously or as authenticated users  
- **Authentication**: JWT-based secure authentication for registered users  
- **User Profile**: View and update user profile
